# 🧠 7z Cheat Sheet

## 🔍 Basic Usage

```bash
7z a archive.7z file1 file2          # Add files to a new archive
7z a archive.7z dir/                 # Add entire directory to archive
7z x archive.7z                      # Extract preserving directory structure
7z e archive.7z                      # Extract all files to current directory (flat)
7z l archive.7z                      # List contents of archive
7z t archive.7z                      # Test integrity of archive
```

## 📦 Supported Formats

```bash
# Create: 7z, xz, bzip2, gzip, tar, zip, wim
# Extract: 7z, xz, bzip2, gzip, tar, zip, wim, ar, arj, cab, chm, cpio,
#          cramfs, dmg, ext, fat, gpt, hfs, ihex, iso, lzh, lzma, mbr,
#          msi, nsis, ntfs, qcow2, rar, rpm, squashfs, udf, uefi, vdi,
#          vhd, vmdk, xar, z
```

## ⚙️ Compression Flags

```bash
7z a -t7z archive.7z dir/            # Force 7z format (default)
7z a -tzip archive.zip dir/          # Create zip instead
7z a -mx=0 archive.7z dir/           # Store only (no compression)
7z a -mx=1 archive.7z dir/           # Fastest compression
7z a -mx=5 archive.7z dir/           # Normal compression (default)
7z a -mx=9 archive.7z dir/           # Ultra compression (slowest, smallest)
7z a -m0=lzma2 archive.7z dir/       # Use LZMA2 method (best for 7z)
7z a -mfb=64 archive.7z dir/         # Fast bytes for LZMA = 64
7z a -md=32m archive.7z dir/         # Dictionary size = 32 MB
7z a -ms=on archive.7z dir/          # Enable solid archive (better ratio)
```

## 🔒 Encryption

```bash
7z a -p archive.7z dir/              # Encrypt with password (prompted)
7z a -pMyPass archive.7z dir/        # Encrypt with inline password
7z a -mhe=on -p archive.7z dir/      # Encrypt filenames too (7z format only)
7z x -pMyPass archive.7z             # Extract with password
```

## 🚫 Exclusions

```bash
7z a archive.7z dir/ -x'!*/.git'           # Exclude .git directories
7z a archive.7z dir/ -x'!*.log'            # Exclude all .log files
7z a archive.7z dir/ -xr'!node_modules'    # Exclude node_modules recursively
7z a archive.7z dir/ -x'!*.tmp' -x'!*.bak' # Exclude multiple patterns
```

## 📁 Extract Options

```bash
7z x archive.7z -ooutput/            # Extract to specific directory (no space after -o)
7z x archive.7z -aoa                 # Overwrite all existing files
7z x archive.7z -aos                 # Skip existing files
7z x archive.7z -aou                 # Auto-rename extracted files if conflict
7z x archive.7z "*.txt"              # Extract only .txt files
```

## 🧰 Maximum Compression Recipe

```bash
# Best ratio for archiving a directory with password + encrypted headers:
7z a \
  -t7z -m0=lzma2 -mx=9 -mfb=64 \
  -md=32m -ms=on -mhe=on \
  -x'!*/.git' -p'eat_my_shorts' \
  archive.7z dir1

# Breakdown:
#   -t7z       7z archive format
#   -m0=lzma2  LZMA2 compression method
#   -mx=9      Ultra compression level
#   -mfb=64    64 fast bytes for LZMA
#   -md=32m    32 MB dictionary size
#   -ms=on     Solid archive (files compressed together)
#   -mhe=on    Encrypt archive headers (hide filenames)
#   -x'!*/.git'  Exclude .git directories
#   -p         Set a password
```

## 🔗 Common Combos

```bash
7z a -ttar -so dir/ | 7z a -t7z -si archive.tar.7z  # Create tar.7z (preserve permissions)
7z l archive.7z | grep "\.sql"                        # Find SQL files in archive
7z x archive.7z -ooutput/ && ls output/               # Extract and verify contents
fd -e 7z | xargs -I {} 7z t {}                        # Test all 7z files in tree
```

## ⚠️ Gotchas

```bash
# 7z does NOT preserve Unix permissions/ownership — use tar.7z for that
# -o has no space: -ooutput/ is correct, -o output/ is WRONG
# Solid archives (-ms=on) compress better but random access is slower
# 7z format does not store symlinks — use tar for repos with symlinks
```
